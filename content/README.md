# Content (working files)

This folder holds **drafts**, **transcripts**, and other working files that are not yet published to the docs site.

- **Drafts:** Articles or tutorials in progress; move to `docs/` when ready.
- **Transcripts:** Video or audio transcripts for accessibility and search.
- **Slides:** Source slides for video tutorials (e.g. for "How stability pools work").

Do not commit large binary files here; use Git LFS or link to external storage if needed. See [CONTRIBUTING.md](../CONTRIBUTING.md) for the content workflow.
