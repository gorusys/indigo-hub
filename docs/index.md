# Welcome to Indigo Protocol Education

Learn about **Indigo Protocol** — a Cardano-based platform for synthetic assets (iAssets) such as **iUSD**, **iBTC**, and **iETH**. This site offers structured learning for everyone from first-time users to developers building on Indigo.

## Choose your path

| If you are… | Start here |
|-------------|------------|
| **New to Indigo or DeFi** | [Introduction to Cardano & DeFi](beginner/intro.md) → [What is a Synthetic Asset?](beginner/synthetic-assets.md) → [Getting Started with Indigo](beginner/using-indigo.md) |
| **Developer** | [Indigo Architecture](developer/arch-overview.md) → [Plutus & eUTxO Primer](developer/plutus-eutxo.md) → [Indigo TypeScript SDK](developer/sdk-setup.md) |
| **Researcher / advanced** | [Advanced Contract Interaction](developer/advanced-contract-interaction.md), [Staking & Governance](beginner/staking-governance.md), and protocol deep dives |

## What you'll learn

- **Novice:** What synthetic assets and CDPs are, how to use the Indigo app, mint iUSD, and manage positions. You’ll understand basic risks (e.g. liquidation) and best practices.
- **Developer:** How Indigo’s contracts and eUTxO model work, how to use the TypeScript SDK, and how to build scripts or dApps that interact with Indigo.
- **Researcher:** Protocol design, tokenomics, stability mechanisms, and governance (INDY token, DAO).
