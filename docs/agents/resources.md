# Indigo Agents — Repos & SDK

Central list of **official Indigo Protocol** repositories for MCP, AI tools, and the SDK. Use these to fetch Indigo intelligence, build bots, or integrate with your own stack.

## Portal and docs

| Resource | Description |
|----------|-------------|
| [Indigo Agents Portal](https://agents.indigoprotocol.io/) | Main site: setup, docs, skills, and API reference |
| [Setup MCP](https://agents.indigoprotocol.io/setup) | Official MCP setup guide |
| [Browse Skills](https://agents.indigoprotocol.io/skills) | All 13 skills for AI agents |

## GitHub repositories

| Repo | Purpose |
|------|---------|
| [IndigoProtocol/indigo-mcp](https://github.com/IndigoProtocol/indigo-mcp) | MCP server — connect LLMs to Indigo Protocol data (57 tools) |
| [IndigoProtocol/cardano-mcp](https://github.com/IndigoProtocol/cardano-mcp) | MCP server for Cardano (6 tools) |
| [IndigoProtocol/indigo-ai](https://github.com/IndigoProtocol/indigo-ai) | AI-related tooling and integrations for Indigo |
| [IndigoProtocol/cardano-ai](https://github.com/IndigoProtocol/cardano-ai) | AI-related tooling for Cardano |
| [IndigoProtocol/indigo-sdk](https://github.com/IndigoProtocol/indigo-sdk) | Indigo TypeScript/JavaScript SDK for contracts and integrations |

Install via npm/npx:

- **indigo-mcp:** `npx @indigo-protocol/indigo-mcp` or `npx @indigo-protocol/indigo-mcp setup`
- **cardano-mcp:** `npx @indigo-protocol/cardano-mcp`
- **Skills bundle:** `npx @indigoprotocol/indigo-skills`
