# Developer Track

This track is for **developers** who want to build on Indigo: understand the architecture, use the TypeScript SDK, and integrate Indigo into scripts or dApps.

## Modules

1. **[Indigo Architecture](arch-overview.md)** — Indigo V2 protocol, Plutus contracts, and high-level design.
2. **[Plutus & eUTxO Primer](plutus-eutxo.md)** — Cardano’s eUTxO model and Plutus basics.
3. **[Indigo TypeScript SDK](sdk-setup.md)** — Install and use the SDK for CDP and iAsset operations.
4. **[Code Lab - Mint iUSD](code-lab-mint-iusd.md)** — Hands-on: mint iUSD via code.
5. **[Advanced Contract Interaction](advanced-contract-interaction.md)** — Liquidations, querying CDP state, and edge cases.
6. **[Integration Example](integration-example.md)** — Build a simple dApp or bot that uses Indigo.
7. **[Testing and Deployment](testing-deployment.md)** — Testnet, testing strategies, and deployment tips.

Work through them in order if you are new to Cardano or Plutus; otherwise you can jump to the SDK and code lab.
